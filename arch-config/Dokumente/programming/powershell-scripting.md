# Powershell scripting

Use PowerShell ISE to develop PowerShell scripts  

## Comparison operators

| operator | description          |
| -------- | -----------          |
| -eq      | equals               |
| -ne      | not equals           |
| -gt      | greather than        |
| -ge      | greater or equals to |
| -lt      | less than            |
| -le      | less or equals to    |
